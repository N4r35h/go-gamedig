#Usage

### as a cli tool

```sh
# <binary> <game_type> <hostname (ip:port)>
go run . samp 127.0.0.1:7777
# or
gogamedig samp 127.0.0.1:7777
```

![example.png](example.png)

### as a library inside your golang project

```sh
go get github.com/N4r35h/go-gamedig
```
